# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-08

### Added
- Initial release of UVis Unity Visualization Runtime
- VegaContainer component with Canvas2D and WorldSpace3D modes
- Bar, Line, and Point mark renderers
- Linear, Log, and Band scale implementations
- Filter, Aggregate, Sort, and Bin data transforms
- Axis and Legend rendering with TextMeshPro
- Custom editor with JSON editing and templates
- Sample chart specifications
